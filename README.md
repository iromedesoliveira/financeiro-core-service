# Financeiro Core Service

Projeto desenvolvido com Java 21 e Spring Boot 3 para gestão de transações financeiras.

## Tecnologias utilizadas:
- Java 21
- Spring Boot 3 (JPA, Web, Data)
- Oracle Database
- Lombok (para redução de boilerplate)

## Estrutura do Projeto:
- **Domain**: Entidades mapeadas com JPA.
- **Repository**: Interfaces para acesso ao banco de dados.
- **Service**: Regras de negócio e processamento.
- **Controller**: Exposição dos endpoints da API.
